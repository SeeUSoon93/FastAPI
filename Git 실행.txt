* Git을 실행하려면 파일을 관리할 폴더를 초기화해야 한다.
* 초기화 하면 해당 폴더 내의 모든 파일을 추적하고 관리할 수 있다.
* 새로운 Git 저장소를 초기화하려면 폴더를 생성한 후 해당 폴더에서 다음 명령 실행
-------------------------------------------------------------------------
$ git init
-------------------------------------------------------------------------

* 파일을 추적하려면 해당 파일을 저장소에 추가하고 커밋해야함.
-------------------------------------------------------------------------
$ git add hello.txt
-------------------------------------------------------------------------
$ git commit -m "Initial commit"
-------------------------------------------------------------------------

* 파일이 변경되면 다음 명령을 사용해 상태 추적
-------------------------------------------------------------------------
$ git status
-------------------------------------------------------------------------

* 파일의 변경 내용을 보려면 다음 명령어
-------------------------------------------------------------------------
$ git diff hello.txt
-------------------------------------------------------------------------

* .gitignore 파일에는 Git의 관리 대상에서 제외할 파일이나 폴더를 지정
* 다음의 명령어로 .gitignore 파일 생성
-------------------------------------------------------------------------
$ touch .gitignore
-------------------------------------------------------------------------

* 다음과 같은 명령어로 .gitignore 파일에 추가
-------------------------------------------------------------------------
$ echo ".env" >> .gitignore
-------------------------------------------------------------------------
